## Poznámky
Nejsem si jistej, u [Arizona.md](songs/arizona.md), jak má být poslední řádek refrému.  
Při tisku jako kniha se musí dát pozor na to, aby pokud bude písnička na více stran tak, že se bude muset během hraní otočit strana, tak aby na nové straně byly akordky ke sloce. U většiny písniček jsou totiž akordy pouze u první sloky.
